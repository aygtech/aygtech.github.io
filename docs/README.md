### 像数 1, 2, 3 一样容易

``` bash
# 安装cli
cnpm i -g @weexbox/cli

# 新建一个weex工程
weexbox create project-name
# 或者
# 新建一个weex和flutter混合工程
weexbox create project-name -f

# 进入工程
cd project-name

# 安装依赖
cnpm i

# 开始写bug
```

::: danger
请务必使用[cnpm](https://npm.taobao.org/)安装@weexbox
:::

::: danger 注意
请确保你的 Node.js 版本 >= 10。
:::
